The Spreadsheet

You can view a demo version of the Audit Spreadsheet here or create a copy to track your own guild! If you want to track your own guild you will first need a key code, which you can get here. The spreadsheet should never be edited, if you wish to add or remove characters you can do it using this page on the website. Characters will then appear in the spreadsheet automatically.
